Automatic Iriigation system

Description:
	This project can be used in IoT project in agriculture domain to identify whether agricultural fields required irrigation or not. 
Requirements:
Basic Requirements to run this projects are:
	Python 2.7 or higher version 
Packages: numpy , pandas, sklearn, matplotlib
Instructions:
For real time prediction field’s data can be collected using DHT11 or DHT12 soil moisture sensor, Humidity and temperature sensor. Apart from this Crop types, Crop days (Days past after sowing) ,is also required for batter understanding the input and output of the system, go through the description of the datasets.
